Detailed Description

An abstract interface for hardware that implements the Universal Asynchronous Receiver Transmitter (UART) hardware communication Protocol.

◆ Flush()

virtual void sjsu::Uart::Flush ( )
inlinevirtual

Will flush all bytes currently head with the UART peripherals buffers.

The default implementation reads out all of the bytes by checking HasData() repeatedly and use Read() to read each. Some implementations may have more efficient methods of clearing their buffers.

Reimplemented in sjsu::lpc40xx::UartBase, and sjsu::stm32f10x::UartBase.

virtual size_t sjsu::Uart::Read ( std::span< uint8_t >  data)
pure virtual

Retrieve bytes from the RX line. If no data is available, this method should return early.

Parameters
data- buffer to load bytes from the uart RX line.
Returns
number of bytes read into the data buffer. Will be less than or equal to the number of bytes passed by size. More bytes can exist in the uart's buffer if the returned size is equal to the passed size.

Implemented in sjsu::lpc40xx::UartBase, and sjsu::stm32f10x::UartBase.
